Политика конфиденциальности Superflight

Наше приложение использует рекламу (CAS.AI Clever Ads Solutions) и может собирать некоторые данные для её показа.

1. Какие данные собираются?

  • Рекламный идентификатор (Advertising ID) — для персонализации рекламы.
  • Данные устройства (модель, версия ОС) — для совместимости с рекламными сетями.
  • IP-адрес — для геотаргетинга рекламы.

2. Как используются данные?

  • Показ релевантной рекламы (баннеры и полноэкранные объявления).
  • Анализ эффективности рекламных кампаний.
  • Данные не передаются третьим лицам, кроме рекламных сетей CAS.AI Clever Ads Solutions.

3. Безопасность

Мы не храним ваши личные данные (email, имя, фото).
